# Shed Before Boarding

**No fur allowed in the cabin!**

A World of Warcraft addon that helps Druids and Shamans quickly cancel their shapeshift forms when using flight masters.

## Features

- Detects when you open a flight master while in a shapeshift form
- Displays a dark overlay on the flight map with a "Cancel Form" button
- One click cancels your form so you can take the flight path
- Works with all Druid forms (Bear, Cat, Travel, Moonkin, Tree, Flight forms)
- Works with Shaman Ghost Wolf form

## Installation

1. Download the latest release
2. Extract to your `Interface/AddOns` folder
3. Restart WoW or reload your UI

## Compatibility

- WoW: The Burning Crusade Classic (Anniversary Edition)
- Interface Version: 20505
